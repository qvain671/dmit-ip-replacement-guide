## 先确认：DMIT 的 IP 真的被封了吗？

很多人遇到连不上就直接认定是 IP 被墙，其实不一定。VPS 死机、SSH 端口被封、服务没有正常跑——这几种情况的症状跟 IP 被墙长得几乎一模一样。

判断的逻辑很简单：**国外能访问，国内不能访问，才是 IP 被墙的特征。**

检测步骤如下：

1. 先去 DMIT 控制面板确认 VPS 状态显示 Running，排除宕机可能
2. 用国内的端口检测工具（如 ping.chinaz.com 或 port.ping.pe），输入你的 IP 和 SSH 端口，检测国内连通性
3. 同一个 IP 和端口，再用国外节点检测一次
4. 国内不通 + 国外通 → **IP 被墙**，该申请换 IP 了
5. 国内国外都不通 → 先检查 VPS 是否正常运行，或 SSH 服务是否挂掉

判断出来是 IP 被封之后，接着看 DMIT 的换 IP 规则。

## DMIT 官方换 IP 政策全解析

DMIT 不同套餐系列，换 IP 的规则差别挺大，别看错了。

### Pro 与 Eyeball（EB）系列：满足条件可免费换

这是用 CN2 GIA 和 CMIN2 优化线路的主力套餐，换 IP 的免费条件如下——四个条件必须**同时满足**：

- 实例已激活超过 7 天（第 8 天起才符合条件）
- 距离上次换 IP 已超过 15 天（包括付费申请的那次）
- ICMP 和 TCP **所有端口**都已被封禁（注意：如果还有端口能通，就不算完全被封，免费申请不受理）
- 实例剩余有效期不少于 7 天

四条都满足，提工单就能免费换。

不满足，或者急着要换：**随时可以付 $5 申请**，每次间隔不少于 7 天。

还有一种特殊情况——如果你刚激活实例，发现 IP 一开始就是封的，这个可以直接提工单联系客服，DMIT 会免费帮你换掉。

### sPro 系列（高防线路）：换 IP 贵

sPro 主打 CFMT DDoS 防护，IP 资源更稀缺。没有 IP Care+ 的情况下，每次换 IP **$15**，间隔 ≥ 30 天。这个成本比较高，如果对防封有需求，选这个系列要有心理准备。

### T1 系列（国际线路）：有 IP Guarantee+ 保底

T1 是国际普通线路，对国内的访问质量不做特别保证。月付套餐默认不保证 IP 在国内可用，可以花 $1 开通 **IP Guarantee+** 服务——一旦 IP 在国内不可用，可以提工单免费重新分配。

非月付（季付及以上）套餐，IP Guarantee+ 已经包含在内，不需要额外购买。

没有 IP Guarantee+ 的其他情况：换 IP $5/次，间隔 ≥ 7 天。

## 申请换 IP 的操作流程

知道了条件，实际怎么申请？

1. 登录 DMIT 控制台（客户端后台）
2. 点击右上角「提交工单（Submit Ticket）」
3. 部门选择技术支持，说明情况：IP 被封，已满足免费换 IP 条件（或表示愿意付费换）
4. 填上被封 IP、实例 ID，附上检测截图更快通过审核
5. 等待客服回复，DMIT 支持中文工单，通常半天内会有回应

> 重要提示：如果是激活时就已经被封的 IP，官方要求**在客服处理期间不要对实例做任何操作**，否则可能需要付费。

## IP 老是被封？选套餐前这几点要想清楚

换 IP 是应急手段，治标不治本。如果你发现 DMIT 的 IP 被封频率很高，问题往往出在套餐和使用方式的匹配上。

**一、套餐系列选对**

实际用下来，Premium（Pro）系列的 IP 被封频率比 T1 低得多。原因不复杂：T1 走普通国际线路，IP 池资源相对混杂；Pro 和 EB 走 CN2 GIA / CMIN2 这类高端优化线路，IP 质量更干净。

如果对国内访问稳定性要求高，预算允许的话，洛杉矶或香港的 Pro 系列是更稳健的选择。

**二、计费周期别只选月付**

月付套餐默认不保证 IP 可用性（尤其是 T1 系列），而且一旦 IP 被封，免费换 IP 的冷却期是 15 天——这段时间机器基本废掉了。

季付及以上套餐，IP Guarantee+ 是内含的，不用额外出钱，保障好一些。

**三、被封之后别急着换**

DMIT 官方有过一个建议：如果 IP 频繁被封，换了新 IP 也可能很快再被封，因为同网段的 IP 处于高风险状态。换之前可以等几天，观察一下是不是大范围的问题。

## 换 IP 之后，怎么降低再次被封的概率？

换完新 IP，这几个习惯能帮你撑更久：

1. **换掉默认 SSH 端口**，22 端口太显眼，换成 50000 以上的随机端口
2. **开启防火墙，只放行必要端口**，不开放不用的服务
3. **单台 VPS 不要同时跑大量代理节点**，流量特征太明显容易触发检测
4. **避免在高峰时段跑大流量业务**，这段时间检测强度也更高

说实话，CN2 GIA 的 IP 不是"免疫"被封的，只是概率低一些。线路质量好不等于永远不会被封，只是轮到你的周期更长。DMIT 的 Pro 系列用下来，连续稳定几个月是常见的，但不是永久保险。

## 常见问题（FAQ）

**Q：DMIT 的 IP 被封，多久能免费换一次？**

Pro 和 EB 系列：每 15 天可申请一次免费换 IP，条件是四个条件同时满足（激活超7天、上次换IP距今超15天、ICMP和TCP全封、剩余期超7天）。急着要换可以随时付 $5。

**Q：刚买的 DMIT VPS，IP 激活就不能用，怎么办？**

直接提工单告知客服，DMIT 对新激活时即被封的 IP 提供免费更换服务，不需要等 7 天。

**Q：T1 套餐的 IP 更容易被封吗？**

T1 是普通国际线路，DMIT 官方不保证该系列在国内的可访问性。月付套餐需要额外购买 IP Guarantee+（$1）才能保障初始 IP 可用；季付及以上套餐已包含 IP Guarantee+。对国内访问稳定性有要求的，优先考虑 Pro 或 EB 系列。

**Q：DMIT 的 IP 能不能随便换，换多少次都行？**

不行。免费换 IP 有频率限制，而且申请条件比较严格（必须全封才算）。付费换也有间隔要求（≥ 7 天）。如果 IP 是被 DDoS 打的，不在免费换 IP 范围内。

**Q：香港 Pro 套餐比洛杉矶 Pro 被封频率低吗？**

物理距离更近，绕路更少，延迟也更低（20-50ms）。IP 被封的情况存在，但用起来延迟低、线路稳，整体体验通常比洛杉矶的更直接。价格也更贵，如果预算允许，香港 Pro 是国内用户的首选。

**Q：DMIT 的 VPS 不满意可以退款吗？**

可以。3 天内使用流量不超过 30GB 的情况下全额退款；1 个月内可申请按剩余价值退款（注意：IP 被墙属于"滥用"情形，可能影响退款资格，建议购买前仔细阅读 TOS）。
